# dsynths-website
 
dSynths Website

## Getting started
### To get this project running locally follow the steps:

- Run
    ```bash
    npm install # npm i
    # to run dev server
    npm run dev
    # to build and serve
    npm run start
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
